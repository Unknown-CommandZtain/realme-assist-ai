Safe System Debloating 🛠️ Removing unwanted system apps (bloatware) can improve battery life and free up memory. You can do this safely without root or breaking your device warranty. Modern Method (Recommended) The easiest way today is using Shizuku combined with Canta directly on your phone, completely eliminating the need for a computer. • Set up Shizuku using Wireless Debugging in Developer Options. • Install Canta from F-Droid or GitHub, link it to Shizuku, and safely uninstall background bloatware. The Traditional PC Method (ADB) 1. Install ADB on your computer using an official platform tools guide. 2. Enable USB Debugging inside your phone's Developer Options and plug it into the computer. 3. Open terminal/command prompt, type adb shell, and authorize the connection prompt on your screen. ⚠️ Critical Warning: Never remove apps critical to system functionality (like core system UIs or security frameworks) or your phone will bootloop. 4. To uninstall a specific app, use the command: pm uninstall -k --user 0 PACKAGE_NAME Example for Facebook: pm uninstall -k --user 0 com.facebook.katana
